Welcome to Make Progress
This is a progress tracking app.
It is intended to help the user visualise their progress in a given area they have difficulty with.
Templates
Templates is where the structure is created for things that you want to keep a track of.
Wheels
Wheels is where you input your confidence value each day, and visualize your progress. The radar will display each entry as a different random colour. This will allow you to see your progess clearly.
Instructions:
  • Navigate to Templates, to either view existing templates or create a new one
  • Navigate to Wheels, to either view progress or input values of your progress
